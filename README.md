## 🚀 Deploy on GitHub Pages (Free Hosting)

1. **Create** a new GitHub repository named `<your-username>.github.io`
2. **Upload** `index.html`, `README.md`, and `LICENSE` to the repository
3. Go to **Settings → Pages**
4. Under *Source* select **Deploy from a branch**
5. Choose **main** branch → **/ (root)** → click **Save**
6. Wait ~60 seconds, then visit `https://<your-username>.github.io`

> ✅ The file is already named `index.html` — GitHub Pages will serve it automatically.
